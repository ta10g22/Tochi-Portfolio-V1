# Tochi's Portfolio

A personal portfolio website showcasing my software, machine-learning, embedded-systems, and digital-hardware projects.

This is the first version of the site. It is built from scratch with HTML and CSS as an introduction to front-end development and UI/UX design.

## Features

- Personal introduction and education overview
- Links to GitHub, LinkedIn, YouTube, Instagram, and email
- Technical skills section
- Software and hardware project summaries
- Project technology icons
- Contact details and a contact-form interface
- Smooth scrolling between page sections

## Technologies

- HTML5
- CSS3
- Google Fonts
- SVG icons
- Git and GitHub

No JavaScript or front-end framework is currently used.

## Run locally

Clone the repository:

```bash
git clone https://github.com/ta10g22/Personal_website.git
cd Personal_website
```

Open `index.html` directly in a browser, or serve the directory with a local web server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Current limitations

- The contact form is presentational and does not yet send messages.
- Project cards do not yet link to source repositories or live demos.
- Mobile responsiveness and accessibility require further refinement.
- The site does not currently use automated tests or a build process.
